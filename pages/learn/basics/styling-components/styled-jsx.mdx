## CSS-in-JS using styled-jsx

If you prefer to write CSS inside JS (a technique called CSS-in-JS), Next.js has built-in support for that. It’s called `styled-jsx` ([documentation](https://github.com/zeit/styled-jsx)).

Suppse that you want to center-align the heading on the index page. Modify `pages/index.js` like so:

```jsx
import PostLink from '../components/PostLink';

const Index = () => (
  <div>
    <h1>Index</h1>
    <p>
      <PostLink href="/posts/first-post">First Post</PostLink>
    </p>
    <style jsx>{`
      h1 {
        text-align: center;
      }
    `}</style>
  </div>
);

export default Index;
```

The text "Index" (which is in `<h1>`) should now be center aligned.
