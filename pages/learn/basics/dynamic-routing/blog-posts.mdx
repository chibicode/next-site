## Displaying a List of Blog Posts

First, let’s create a file to store information about each blog post:

- Create the `lib` directory (on the top level of the project folder)
- Add `posts.js` inside with the following contents:

```js
export const postsData = {
  'learn-typescript': {
    title: 'How to learn TypeScript'
  },
  'learn-graphql': {
    title: 'How to learn GraphQL'
  }
};

export const postsKeys = ['learn-graphql', 'learn-typescript'];
```

- `postsData` contains data about each blog post. It stores post data such as `title`.
- `postsKeys` is a list of `postsData` keys. We’ll display our bog posts in this order.

### Update the index page

Let’s display the post names on the index page. Modify `pages/index.js` as follows:

```jsx
import Link from 'next/link';
import { postsData, postsKeys } from '../lib/posts';

const Index = () => (
  <div>
    <h1>Index</h1>
    <ul>
      {postsKeys.map(key => (
        <li key={key}>{postsData[key].title}</li>
      ))}
    </ul>
    <p>
      <Link href="/about">
        <a>About</a>
      </Link>
    </p>
  </div>
);

export default Index;
```

If you open [http://localhost:3000](http://localhost:3000), you should now see:

- How to learn GraphQL
- How to learn TypeScript
